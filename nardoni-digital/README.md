# Nardoni Media

Fill Your Calendar with Customers in 30 Days

## About

Nardoni Media helps local businesses attract customers through targeted digital marketing. We specialize in Google Ads, Facebook/Instagram Ads, and SEO to put your business in front of locals actively searching for your services.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/barecloudz/nardoni-digital.git

# Navigate to the project directory
cd nardoni-digital

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Deployment

Build the project for production:

```sh
npm run build
```

The built files will be in the `dist` directory.
